# Задание: распределение городов по государствам

## Описание алгоритма
Дан связный граф дорог. Заданы k столиц. На каждом шаге находится ближайший незанятый город к любому из государств (по минимальному расстоянию от этого города до любого города, уже принадлежащего государству). Город присоединяется к этому государству. Процесс продолжается, пока все города не распределены.

## Как запустить
```bash
gcc main.c -o program.exe
program.exe < input.txt
